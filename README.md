## Create

### new vite project with dependencies
```shell
npm create vite@latest boilerplate -- --template vanilla
```
### dependencies
```shell
npm install --save-dev sass
npm install bootstrap @popperjs/core
```

## Usage

### start dev server
```shell
npm run dev
```

### build for production
```shell
npm run build
```

### locally preview production build
```shell
npm run preview
```
